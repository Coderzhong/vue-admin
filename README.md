# 基于vue的admin前端框架

预览地址：https://coderzhong.github.io/vue-admin/dist/

## 介绍
- clone本项目后可以直接基于此直接进行前端业务开发
- 登录、登出、JWT、管理token过期时间、记住登录状态
- 前端鉴权，实现页面权限管理和组件权限管理（指令封装）
- 递归渲染侧边栏导航
